# 菏泽家乡旅行推荐 App / 허쩌 고향 여행 추천 앱
이것은 기말 과제 요구사항에 부합하는 Flutter 애플리케이션 예시로, 주제는 「중국 모란의 도시 허쩌 여행 추천」입니다.

## 포함된 내용 / 已包含内容
- 고향 지역 소개：허쩌 위치, 도시 특징, 모란 문화
家乡地区介绍：菏泽位置、城市特色、牡丹文化
- 대표 관광 명소：조주 모란원, 조주 고성, 천향 공원
代表旅游景点：曹州牡丹园、曹州古城、天香公园
- 대표 미식：산현 양고기 국, 조현 병빵, 운성 장몽, 모란 과자
代表美食：单县羊肉汤、曹县烧饼、郓城壮馍、牡丹糕点
- 추천 여행 루트：당일치기 여행 코스
推荐旅行路线：一日游路线
- 이미지 및 페이지 구성：하단 네비게이션, 카드, 아이콘, 화성 테마 색상
图片及页面构成：底部导航、卡片、图标、花城主题色
- AI 활용 내용：문안 정리, 페이지 디자인 참고, 루트 기획 설명 초안 작성
AI 使用内容：文案整理、页面设计、路线规划说明
- 프로젝트 수행 소감：AI 설명 페이지 하단에 기재
项目执行心得：放在 AI 说明页面底部

## 실행 방법 / 运行方法
Flutter 환경이 설치된 컴퓨터에서 해당 폴더로 진입 후 아래 명령어 실행：
在安装 Flutter 的电脑上进入本文件夹，执行：
```bash
flutter pub get
flutter run
```

## APK 빌드 / 生成 APK
```bash
flutter build apk --release
```

빌드 파일 경로 / 生成位置：
```text
build/app/outputs/flutter-apk/app-release.apk
```
