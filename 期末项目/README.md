# 菏泽家乡旅行推荐 App

这是一个符合期末项目要求的 Flutter 应用示例，主题为“菏泽，中国牡丹之都旅行推荐”。

## 已包含内容

- 家乡地区介绍：菏泽位置、城市特色、牡丹文化
- 代表旅游景点：曹州牡丹园、曹州古城、天香公园
- 代表美食：单县羊肉汤、曹县烧饼、郓城壮馍、牡丹糕点
- 推荐旅行路线：一日游路线
- 图片及页面构成：底部导航、卡片、图标、花城主题色
- AI 使用内容：文案整理、页面设计、路线规划说明
- 项目执行心得：放在 AI 说明页面底部

## 运行方法

在安装 Flutter 的电脑上进入本文件夹，执行：

```bash
flutter pub get
flutter run
```

## 生成 APK

```bash
flutter build apk --release
```

生成位置：

```text
build/app/outputs/flutter-apk/app-release.apk
```
